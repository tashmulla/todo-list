# 🚀 TODO List Frontend - React & Next.js

## 🌟 Overview
This is a simple frontend application for managing a TODO list. It allows users to add, edit, and delete tasks, as well as mark them as completed.

## ✨ Features
- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 🔍 View tasks by status (completed, pending)

## 🛠️ Technologies Used
- 🌐 HTML
- 🎨 CSS
- ⚙️ JavaScript
- ⚛️ React
- ➡️ Next.js

## 📦 Installation
1. Clone this repository
2. Navigate to the frontend directory:
  ```bash
  cd todo-frontend
  ```
3. Install dependencies:
  ```bash
  npm install
  ```

## 🚀 Local usage
1. Build the project:
  ```bash
  npm build
  ```
2. Start it up:
  ```bash
  npm start
  ```
3. Open your browser and navigate to `http://localhost:3000`.

## 👩‍💻 Local development
1. Start the development server with:
  ```bash
  npm run dev
  ```

2. Open your browser and navigate to `http://localhost:3000`.
